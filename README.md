# 📊 Student Performance Analysis (R)

## 🔍 Overview
This project analyzes how lifestyle factors such as sleep, study time, and extracurricular activities impact student academic performance.

Using a dataset of over 6,600 students, I performed exploratory data analysis (EDA) and statistical comparisons to uncover patterns that influence exam scores.

---

## 🎯 Objectives
- Understand the relationship between sleep and academic performance
- Compare students with and without extracurricular activities
- Analyze how study time impacts exam scores
- Identify key factors that contribute to student success

---

## 🛠️ Tools & Technologies
- R
- tidyverse
- dplyr
- ggplot2

---

## 📁 Dataset
- Source: Kaggle (Student Performance Dataset)
- Size: ~6,600 records

---

## 📊 Key Insights
- Students with consistent sleep schedules tend to perform better on exams
- Moderate extracurricular involvement correlates with higher academic performance
- Study time has diminishing returns after a certain threshold
- Lifestyle balance plays a significant role in student success
